# 🎮 Tic Tac Toe

A console-based **Tic Tac Toe** game built in **Python** to strengthen my understanding of Python fundamentals and Object-Oriented Programming (OOP). This project demonstrates clean code organization, game logic implementation, and problem-solving skills.

## ✨ Features

- 🎯 Interactive command-line gameplay
- 👥 Human vs Human mode
- 🤖 AI opponent using the Minimax algorithm 
- ✅ Input validation
- 🏆 Winner & draw detection
- 🧩 Object-Oriented Programming (OOP)

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming
- Git & GitHub
